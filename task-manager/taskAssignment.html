# 任务分配功能

# 功能描述

任务分配功能是将任务分配给指定的用户，任务分配后，该任务将被添加到该用户的待办任务列表中。

# 流程

  1. 选择任务分配功能
  2. 加载待分配的任务
  • 自动分配
  1. 选择已经编辑好的任务规则
  2. 自动按照该规则分配任务
  3. 手分配任务
  • 手动分配
  1. 选择待分配的任务（样品或者检测，一般情况下是检测）
  2. 选择岗位或者具体人员
  3. 分配任务
